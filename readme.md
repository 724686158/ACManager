# 山东科技大学ACM集训队管理系统

---

### 技术栈

- 前台：JSP、JQuery、Bootstrap
- 后台：Java8、Kotlin、Spring、SpringMVC
- 数据库：MySql、Druid、Hibernate、Spring Data JPA
- 服务器：Jetty、Tomcat
- 开发工具：Maven、IntelliJ IDEA、JRebel
- 算法：Agnes、TrueSkill

### 功能

- 个人uva、cf、队内训练信息
- 在线队内UVA统计功能
- 解析vj榜单
- 管理比赛和集训
- 随机组队赛分队
- 个人、队伍算分评测
